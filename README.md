# CustomerSettlement — программа для автоматизации расчётов с покупателями
**CustomerSettlement** — десктопное приложение для малого бизнеса, автоматизирующее выставление счетов, учёт оплат и формирование закрывающих документов. Решает проблему хаотичного учёта в Excel: позволяет вести единую базу контрагентов и договоров, отслеживать статус оплаты по каждому счёту, автоматически пересчитывать НДС и итоги, а также формировать печатные формы (счёт, акт, акт сверки) в один клик. Внедрение сократило время на выписку счетов с 30 до 3 минут и исключило ошибки при ручном пересчёте сумм.

---

##  Возможности
- **Справочники**: контрагенты, договоры, номенклатура услуг
- **Счета**: создание/редактирование с несколькими позициями, автоматический расчёт НДС и итогов, фильтрация по дате, контрагенту и договору, поиск по номеру или названию.
- **Оплаты**: добавление платежей, автоматическое обновление статуса счёта
- **Печать**: счёт на оплату, акт выполненных работ, акт сверки, возможность сохранения в PDF
- **Отчёты**: статистика по услугам с фильтрацией по периоду и категории

## Технологии
- **C# / .NET 10** — основная платформа
- **WPF** — графический интерфейс
- **Entity Framework Core** — ORM для работы с БД
- **SQLite** — встроенная база данных
- **WebView2** — печать и экспорт в PDF
- **HTML / CSS** — шаблоны печатных документов
  
## Ключевые паттерны
- **MVVM** — разделение UI и бизнес-логики
- **Repository** — абстракция доступа к данным
- **Dependency Injection** — слабосвязанные компоненты
- **Command** — инкапсуляция действий
- **Factory** — создание ViewModel и окон
